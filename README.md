    - Java



## 시작하기

### 실행환경

- yarn

### 설치

프로젝트를 클론합니다.

```
git clone https://github.com/tmdgjs/DailyDiary
```

클론했던 디렉터리로 이동해 yarn 패키지를 설치합니다.

```
yarn
```

### 프론트엔드 개발 서버

개발 서버를 실행합니다.

```
yarn start
```

http://localhost:3000/ 으로 접속하면 실행 가능합니다.



### API 개발 서버

API 서버를 실행해 줍니다.

```
https://localhost:8080
```
